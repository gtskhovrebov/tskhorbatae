# TSKHORBATÆ 2.1 Recovery — инструкция

## Что вернулось
В этой версии возвращены ключевые элементы старой концепции:
- План дня
- Календарь
- Ежедневные задачи
- Галочки выполнения
- Самопроверка
- Экзамен недели

## Ежедневный цикл
Каждый день система сама формирует 5 базовых задач:
1. Осетинский язык: 10 слов.
2. Обработать 3 новости.
3. Обновить 1 актора или досье.
4. Сделать 1 аналитический вывод.
5. Закрыть самопроверку дня.

## Как пользоваться
### 1. Открой «План дня»
Там находятся задачи на сегодня. Отмечай галочкой выполненное.

### 2. Открой «Календарь»
Можно смотреть задачи по дням. Нажатие на день открывает план этого дня.

### 3. Открой «Самопроверка»
В конце дня заполни:
- что сделал;
- доказательство;
- следующий шаг.

Раз в неделю проходи экзамен.

## Словарь
Поле поиска больше не должно сбрасываться после одной буквы.
Пиши обычной клавиатурой. Кнопка `ӕ` просто вставляет букву в поле.

## Как обновить на GitHub Pages
1. Распакуй архив.
2. Замени в репозитории:
   - index.html
   - manifest.json
   - icon-192.png
   - icon-512.png
3. Подожди 1–2 минуты.
4. На iPhone открой сайт в Safari.
5. Удали старую иконку с экрана Домой и добавь заново, чтобы подтянулась новая иконка.
